## Run Locally

* Run `yarn && yarn start`
* App will run on `localhost:8080`

## Base Tools Used

-   Typescript
-   Webpack
-   Eslint
-   Styled Components
-   Husky Hooks
-   Heroku
-   Circleci
-   Docker
-   Nginx for production
-   [Sandbox](https://getsandbox.com/) for smart stubs(/stubs)

## Optimizations

-   gzip compression.
-   webpack bundle minifications.
-   webpack bundle analyzer(runs at port 8888 on localhost)
-   Lazy Loading
-   Code Splitting
